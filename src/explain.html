What is a Flow Network?

A flow network is a directed graph where each node has a capacity and each edge receives a flow. This graph is generally used to model the transportation of resources between locations which have a network of routes with limited capacity, which naturally brings up the problem of maximum flow. Finding out maximum flow to allow for the most efficient routes have real world applications such as in transport, utilities and economics.

The Ford-Fulkerson Algorithm

The Ford-Fulkerson algorithm works by finding augmented paths and calculating flow throughout the routes. It starts with the initial flow of zero. If a capacity is not full along a path, which is found using a graph transversal algorithm such as breath-first search or depth-first search, then the algorithm works by augmenting it through making sure the edge flow does not exceed the given capacity and incoming flow is equal to outgoing flow except for the source and sink vertices. This is done by increasing the flow along a path by its maximum amount that does not violate the rules mentioned before.

The time complexity is O(E × F), where E is the number of edges and F is the maximum flow of the graph.

The space complexity is O(V + E), as we need to track all of the edges and verticies for the residual graph.

The Edmond-Karp Algorithm

The Edmond-Karp algorithm works by finding the augmented paths and calculating flow throughout the routes. It works very similarly to the Ford-Fulkerson algorithm as it is an optimised version of the former algorithm, however it specifies to use a breath-first search as a way of finding paths to be augmented. The use of a breath-first search graph transversal guarantees that the augmented paths found will be the shortest ones, thus saving time from the original algorithm.

The time complexity is then O(VE2), where V is the number of vertices and E is the number of edges.

The space complexity is O(V + E), similarly to Ford-Fulkerson algorithm.
